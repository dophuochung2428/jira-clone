# Jira Clone

A frontend-focused project management application inspired by Jira,
built to practice UI workflows and modern React architecture.

## Overview

The application replicates core Jira features such as workspaces, projects, tasks, and kanban-style workflow management.  
It is built as a personal project to demonstrate frontend development skills.

This project focuses on practicing:

- Component-based architecture
- State management patterns
- Drag & drop UX interactions

## Features

- Responsive Kanban board UI
- Drag and drop task interaction
- Dynamic task state updates
- Client-side form validation
- Reusable UI components

## Tech Stack

- Frontend: Next.js, React, Tailwind CSS
- State Management: React Hooks / Context
- UI Pattern: Kanban board, drag and drop
- Backend: Simple REST API (supporting frontend features)

## Getting Started

### Clone the repository

```bash
# Clone the repository
git clone https://github.com/dophuochung2428/jira-clone.git
cd jira-clone

# Install dependencies
npm install

# Run the development server
npm run dev
```
